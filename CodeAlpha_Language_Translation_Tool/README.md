# Task 1: Language Translation Tool

Real-time language translation web app using Flask + deep-translator (Google Translate).

## Tech Stack
- Python, Flask, deep-translator, langdetect, HTML/CSS/JavaScript

## How to Run
pip install flask flask-cors deep-translator langdetect
python translator_backend.py
# Open http://localhost:5000 in your browser

## Features
- Translates text between 30+ languages instantly
- Auto-detects the input language
- Translates live as you type (no button needed)
- Swap source and target languages with one click
- Copy translated text to clipboard
- Text-to-speech for both input and translated text
- Translation history panel showing last 5 translations
