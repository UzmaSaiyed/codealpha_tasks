# Task 2: FAQ Chatbot

An NLP-powered FAQ chatbot using TF-IDF and cosine similarity.

## Tech Stack
- Python, NLTK, scikit-learn, Flask

## How to Run
pip install nltk scikit-learn flask flask-cors
python faq_chatbot.py
# Open http://localhost:5000

## Features
- Text preprocessing (tokenize, lemmatize, stopword removal)
- TF-IDF vectorization + cosine similarity matching
- Chat UI with confidence scores
