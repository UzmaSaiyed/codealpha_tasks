# Code Alpha - Artificial Intelligence Internship Tasks

A collection of AI and NLP projects built during my internship.

---

## Task 1: Language Translation Tool
A real-time web app that translates text between 30+ languages using Google Translate, with auto language detection and text-to-speech.
- Tech: Python, Flask, deep-translator, HTML/CSS/JavaScript
- Folder: `CodeAlpha_Language_Translation_Tool/`


## Task 2: Chatbot for FAQs
An NLP-powered FAQ chatbot that matches user questions to the most similar FAQ using TF-IDF and cosine similarity.
- Tech: Python, NLTK, scikit-learn, Flask
- Folder: `CodeAlpha_Chatbot_for_FAQs/`


## Task 4: Object Detection and Tracking
Real-time object detection and tracking using YOLOv8 and a SORT-style IoU tracker, with persistent tracking IDs and live FPS display.
- Tech: Python, OpenCV, YOLOv8 (Ultralytics)
- Folder: `CodeAlpha_Object_Detection_and_Tracking/`
